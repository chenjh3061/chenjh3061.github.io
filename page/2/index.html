jQuery是一个快速、简洁的JavaScript库，旨在简化HTML文档遍历与操作、事件处理、动画和Ajax交互，为Web开发提供更简单的API。 

jQuery的核心概念 

1. 选择器（Selectors） 

选择器是jQuery的核心功能之一，用于查找和选取HTML元素。它们基于CSS选择器，但jQuery提供了更多的功能。 

// 选择所有

元素 $(“p”) 

// 选择id为”myId”的元素 $(“#myId”) 

// 选择class为”myClass”的所有元素 $(“.myClass”) 

// 选择所有

元素中的第一个 $(“p:first”)  

2. 事件处理（Event Handling） 

jQuery使得事件处理变得简单。你可以使用.on()方法来绑定事件处理程序。 

// 点击事件 $(“button”).on(“click”, function() {   alert(“按钮被点击了”); });  

// 鼠标移入和移出事件 $(“div”).on(“mouseenter mouseleave”, function() {   $(this).toggleClass(“hover”); });  

  1. DOM 操作（DOM Manipulation）

 

jQuery提供了许多方法来操作DOM元素，如添加、删除、修改元素等。 

// 添加类 $(“p”).addClass(“myClass”);  

// 移除类 $(“p”).removeClass(“myClass”);  

// 设置文本内容 $(“#myId”).text(“新文本内容”);  

// 插入元素 $(“ul”).append(“

    $(“#left-well”).parent().css(“background-color”, “blue”)

    4. 动画（Animations） 

    jQuery有一些内置的简单动画方法，如 .hide()、.show()、.slideUp()、.slideDown() 和 .fadeIn()。 

    // 淡入元素 $(“#myId”).fadeIn();  

    // 滑动隐藏元素 $(“.myClass”).slideUp();  

    // 切换显示/隐藏 $(“p”).toggle();  

    1. Ajax

     

    jQuery简化了Ajax请求，使得与服务器进行异步数据交互变得更简单。 

    // 简单的GET请求 $.get(“https://api.example.com/data“, function(data) {   console.log(data); });  

    // 使用$.ajax()进行复杂请求 $.ajax({   url: “https://api.example.com/data“,   type: “POST”,   data: { name: “John”, age: 30 },   success: function(response) {     console.log(response);   },   error: function(xhr, status, error) {     console.error(error);   } });  

    6.禁用元素 

    能用jQuery改变HTML标签的非CSS属性 

    //禁用按钮 

    $(“button”).prop(“disabled”, true”)

    性能和最佳实践 

    尽管jQuery提供了方便的API，但在性能和代码质量方面也有一些最佳实践： 

    1. 缓存选择器：避免重复的DOM查询。  var $myElement = $(“#myElement”); $myElement.text(“新的文本”); 

    2. 链式调用：简化代码并提高性能。  $(“#myElement”).css(“color”, “red”).hide().fadeIn(); 

    3. 事件委托：特别是在动态添加元素时，使用事件委托来提高性能。  $(“ul”).on(“click”, “li”, function() {   $(this).toggleClass(“active”); }); 

    4. 减少DOM操作：尽量减少直接的DOM操作，可以先构建好HTML字符串或使用文档片段。  var newContent = “

      “; $(“#container”).html(newContent);

     

    结束语 

    jQuery虽然在现代JavaScript框架（如React、Vue和Angular）兴起后其使用率有所下降，但由于其简洁的API和广泛的浏览器兼容性，仍然在许多项目中发挥重要作用。无论你是新手还是有经验的开发者，掌握jQuery都将对你的Web开发技能有很大的提升。

    正则表达式用于匹配指定的字符串。JavaScript 中有多种使用正则表达式的方法。 测试正则表达式的一种方法是使用 .test() 方法。 .test() 方法会把编写的正则表达式和字符串（即括号内的内容）匹配，如果成功匹配到字符，则返回 true，反之，返回 false。 

    字符串中查词： 

    Let myString  = “12345678” 

    Let myRegex = /123/ 

    Let result = myRegex.test(myString) 

    中间可以加 | 表示或者，返回值是布尔，//右边加上i表示忽略大小写，加g表示全局搜索(否则只搜索一次)。 

    let extractStr = “Extract the word ‘coding’ from this string.”; 

    let codingRegex = /coding/;  

    let result = extractStr.match(codingRegex);  

    如果是match方法，可以提取找到的实际匹配项，由字符串调用，和test反过来。 

    匹配返回具体值用match，其他用test。 

    通配符 . 可以表示匹配任意的字符， 

    使用字符集 （character classes）更灵活的匹配字符。 把字符集放在方括号（[ 和 ]）之间来定义一组需要匹配的字符串。例如，如果想要匹配 bag、big 和 bug，但是不想匹配 bog。 可以创建正则表达式 /b[aiu]g/ 来执行此操作。 [aiu] 是只匹配字符 a、i 或者 u 的字符集。如果是a到e就写[a-e]，配合数字就写成/[a-z0-9]/ig。 

    创建一个不想匹配的字符集合称为否定字符集（ negated character sets）。 

    要创建否定字符集，需要在开始括号后面和不想匹配的字符前面放置脱字符（即^）。 

    例如，/[^aeiou]/gi 匹配所有非元音字符。 注意，字符 .、!、[、@、/ 和空白字符等也会被匹配，该否定字符集仅排除元音字符。 

    至少出现一次的字符用 + 匹配，至少零次的就 * 。 

    惰性匹配： ？ 尽可能多的匹配。

    #性能指标 

    • 💬 描述：用于衡量一个 Web 页面的性能。 

    • 📚 资源： https://juejin.cn/post/6844904153869713416(opens new window) 

    • FP（First Paint） 

      • 💬 描述：从开始加载到浏览器首次绘制像素到屏幕上的时间，也就是页面在屏幕上首次发生视觉变化的时间。

       

    • FCP（First Contentful Paint） 

      • 💬 描述：浏览器首次绘制来自 DOM 的内容的时间。

       

    • FMP（First Meaningful Paint） 

      • 💬 描述：页面的主要内容绘制到屏幕上的时间。

       

    • FSP（First Screen Paint） 

      • 💬 描述：页面从开始加载到首屏内容全部绘制完成的时间，用户可以看到首屏的全部内容。

       

    • TTI（Time to Interactive） 

      • 💬 描述：表示网页第一次完全达到可交互状态的时间点，浏览器已经可以持续性的响应用户的输入。

    JavaScript 提供了多种内置的数据结构，这些数据结构可以帮助开发者有效地管理和操作数据。以下是 JavaScript 中主要的数据结构及其相关特性和用法： 

    1. 原始类型（Primitive Types）

     

    • Number: 表示整数和浮点数。 const num = 42; const pi = 3.14;  

    • String: 表示字符序列。 const str = “Hello, world!”;  

    • Boolean: 表示真值（true）或假值（false）。 const isTrue = true;  

    • null: 表示一个空值。 const empty = null;  

    • undefined: 表示未定义的值。 let notDefined;  

    • Symbol: 表示唯一的标识符。 const sym = Symbol(‘description’);  

    • BigInt: 表示任意精度的整数。 const bigInt = 1234567890123456789012345678901234567890n;

      

    1. 对象类型（Object Types）

     

    • Object: 一个键值对的集合。 const obj = { name: “Alice”, age: 30 };  

    • Array: 有序元素的集合。 const arr = [1, 2, 3, 4, 5];  

    • Function: 可执行的代码块。 function greet() { console.log(“Hello!”); }

      

    1. 特定用途的数据结构

     

    • Set: 一种数据结构，它只存储唯一值。 const set = new Set([1, 2, 3, 3, 4]); console.log(set); // 输出: Set { 1, 2, 3, 4 }  

    • Map: 一种数据结构，它存储键值对，键可以是任何类型。 const map = new Map(); map.set(‘name’, ‘Alice’); map.set(‘age’, 30); console.log(map.get(‘name’)); // 输出: Alice  

    • WeakSet: 类似于 Set，但其成员必须是对象，并且这些对象都是弱引用，可以被垃圾回收。 const weakSet = new WeakSet(); let obj = {}; weakSet.add(obj);  

    • WeakMap: 类似于 Map，但其键必须是对象，并且这些键都是弱引用，可以被垃圾回收。 const weakMap = new WeakMap(); let obj = {}; weakMap.set(obj, ‘value’);

      

    1. 特殊用途的数据结构

     

    • TypedArray: 用于处理二进制数据的一组对象，包括 Int8Array, Uint8Array, Uint8ClampedArray, Int16Array, Uint16Array, Int32Array, Uint32Array, Float32Array, 和 Float64Array。 const int8 = new Int8Array(4); int8[0] = 42;  

    • ArrayBuffer: 用于处理固定长度的原始二进制数据。 const buffer = new ArrayBuffer(16); const view = new Uint8Array(buffer); view[0] = 42;  

    • DataView: 用于处理 ArrayBuffer 中的不同类型的数据。 const buffer = new ArrayBuffer(16); const view = new DataView(buffer); view.setInt8(0, 42);

      

    1. 日期和时间

     

    • Date: 用于处理日期和时间。 const now = new Date();

      

    1. 正则表达式

     

    • RegExp: 用于匹配文本模式的对象。 const regex = /hello/i;

      

    1. 错误类型

     

    • Error: 表示运行时错误的对象。 try { throw new Error(“Something went wrong”); } catch (e) { console.error(e); }

      

    以上是 JavaScript 中主要的数据结构。它们各自有不同的用途和特性，帮助开发者在不同的场景下有效地管理和操作数据。 

    额外的数据结构和工具 

    1. 栈（Stack）

     

    栈是一种后进先出（LIFO）的数据结构。 

    class Stack { 

     constructor() { 

     this.items = []; } 

     push(element) { 

     this.items.push(element); } 

     pop() { return this.items.pop(); }  

    peek() { return this.items[this.items.length - 1]; }  

    isEmpty() { return this.items.length === 0; }  

    size() { return this.items.length; } 

     clear() { this.items = []; } 

     }  

    1. 队列（Queue）

     

    队列是一种先进先出（FIFO）的数据结构。 

    class Queue {  

    constructor() { this.items = []; }  

    enqueue(element) { this.items.push(element); }  

    dequeue() { return this.items.shift(); }  

    front() { return this.items[0]; }  

    isEmpty() { return this.items.length === 0; } 

     size() { return this.items.length; } 

     clear() { this.items = []; }  

    }  

    1. 双端队列（Deque）

     

    双端队列可以从前后两端添加或移除元素。 

    class Deque {  

    constructor() { this.items = []; }  

    addFront(element) { this.items.unshift(element); }  

    addBack(element) { this.items.push(element); }  

    removeFront() { return this.items.shift(); }  

    removeBack() { return this.items.pop(); }  

    peekFront() { return this.items[0]; }  

    peekBack() { return this.items[this.items.length - 1]; }

    嵌套的对象同理。 

    对象的解构与数组有一个重要的不同。数组的元素是按次序排列的，变量的取值由它的位置决定；而对象的属性没有次序，变量必须与属性同名，才能取到正确的值。如果 没有对应同名属性，就会变成undefined。 

    解构赋值允许默认值， 

    1. let [foo = true] = []; 

    2. foo // true 

    3. let [x, y = ‘b’] = [‘a’]; // x=’a’, y=’b’ 

    4. let [x, y = ‘b’] = [‘a’, undefined]; // x=’a’, y=’b’

     

    注意，ES6 内部使用严格相等运算符（===），判断一个位置是否有值。所以，只有当一个数组成员严格等于undefined，默认值才会生效。 

    1. let [x = 1] = [undefined]; 

    2. x // 1 

    3. let [x = 1] = [null]; 

    4. x // null

     

    注意ES6会把句子开头的{理解成代码块，最好加括号：({x} = {x: 1});

    ![[Pasted image 20240830164724.png]]

    ES6 的规则是，只要有可能导致解构的歧义，就不得使用圆括号。 

    但是，这条规则实际上不那么容易辨别，处理起来相当麻烦。因此，建议只要有可能，就不要在模式中放置圆括号。
