面试人际关系题

考验办事能力，解决一般工作情景下人与人直接发生的问题。

主体多样，问题复杂，一定要工作为重。
![[Pasted image 20241219160656.png]]
对同事：着眼合作多理解，立足长远多承担，分工协作多沟通，相互学习促成长。

对领导：尊重领导权威，适应领导风格，服从领导安排，请示领导意见，完成领导任务，汇报工作结果。

对下属（少）：理解下属有苦衷，主动解其难和蒙，用好长处多鼓励，以身作则常带动。

对群众：放低姿态笑脸迎，人民至上记心中，情理利法解问题，兼顾灵活守原则。

答题思路

（先）看法：态度、认识、原因、影响、原则等

（再）办法：办人办事（现在与将来）

例题：1、你和小王互换轮岗，你来到新部门后，领导经常叫小王来处理本属于你工作职责的事，而只给你安排一些琐事，你怎么办？
![[Pasted image 20241224203903.png]]
2、领导让你和小李共同去完成一项工作，工作很紧急，需要在三天内完成，但小李近期想要休假备考研究生考试，考试一周后开始，你怎么办？
![[Pasted image 20241224203933.png]]
2、小李领导要求很严格，一旦出错就被责备。小李自嘲：在办公室大气不敢出，生怕被责备。如果你是小李，你会怎么做？
![[Pasted image 20250119154010.png]]
3、你是新进单位的工作人员，你们科室科长事必躬亲，做事认真，注重细节，喜欢做事细致的下属；科员老刘工作积极性不高，为人冷淡；科员小王新进单位两年，为人热情，工作积极。请问你该如何与他们三人相处？
![[Pasted image 20241224204312.png]]
4、小李被作为业务骨干抽调到上级部门工作，但是原单位还会给他安排工作任务，有的时候小李就两方面都兼顾不了，如果你是小李，你怎么办？
![[Pasted image 20241224204403.png]]
5、你和小张一起值班，小张因没来受到领导的批评。小张不满，说你只顾自己表现，同事对你议论纷纷，你怎么办？
![[Pasted image 20241224204334.png]]
6、公司有一项任务交给老张，老张说三天就能完成，但是老张因为有事出差，这项工作就交给你来做，结果你一天就完成了，老张知道了这件事后很生气，你怎么做？
![[Pasted image 20241224204837.png]]
7、三个人的一段谈话，分别是：
小王说，当局者迷，旁观者清。发现自身的错误，还需要从领导和同事来进行指正；
小张说，人贵有自知之明，但是人最大的问题是自己不能意识到自己的错误。或者即使意识到了，也会去逃避，不愿面对；小李说，金无足赤，人无完人。每个人都有缺点，每个人都不是完美的，但是只要优点多于缺点就可以了。
请你谈谈对这三句话的理解，并说说自己的不足？

8、领导交给小张一项工作，让小王当副手，可是汇报工作的时候领导只找了小王，还表扬了小王。如果你是小张，你怎么办？
![[Pasted image 20250119161350.png]]

9、你写了一篇文章给领导看，结果渺无音讯。但后来在某杂志上看到一篇文章和你写的文章很像，署名是领导的名字，你怎么办?

![[Pasted image 20250119163912.png]]

4、市民老张到某局办理一项行政审批事项，工作人员小王告诉他，“您的暂时办不了，要过段时间才行”。老张问：“审批权已经下放给市里了，怎么办不了？”小王说：“审批权是下放了，但局里技术条件和人员跟不上，要不您还是到省厅试试？”老张听了很生气：“怎么还让我往省里跑？既然下放了，你就得给我办！”。假如你是小王，遇到这种情况，应怎么办？
