判断推理笔记

国考40题左右，非常重要。要做到90以上准确率。 

四大题型：图形推理(最难)、类比推理、定义判断、逻辑推理(难)。 

二、定义判断

看清提问、抓准信息、比较选项 

抓主客体、原因/条件+结果、方式+目的、包含定义的句子先看、补充说明、同构选项优先排除、 

![[Pasted image 20240831145535.png]]

选项记得挑最好的，多定义和同构选项可以排除。 

![[Pasted image 20240831145553.png]]

三、类比推理

分两词(造句)，三词(关系)，填空(代入)。 

逻辑和对应可以试试造句， 

语义关系（近义反义比喻义象征义、感情色彩、程度轻重、词语本身结构）、多积累成语。 

逻辑关系（全同关系、包容关系（注意种属和组成关系的不同，小心泛指和特指）、

并列关系（矛盾关系、反对关系、并列关系、交叉关系、

3、无法造句的

完整成语诗句已经被拆开、两个概念不相关无法造句、选项高度相似且题干并列近义居多。 

找因果关系、方式-目的、先后顺序。把材料拆开。 

因有自然人工，果有好坏 

二级辨析考虑词性、主语、出身等

四、逻辑判断

翻译推理：

有逻辑关联词的，（抓逻辑关系，注意关键词，逆否推理注意同时肯否，注意推前推后 

后推前：只有才、不。。不、才、除非。。否则。。、。。是。。的必要条件。 

注意：逆否关系四句话：
肯前必肯后，否后必否前，否前不一定，肯后不一定

![[Pasted image 20240831145659.png]]
![[Pasted image 20240831150756.png]]
需要，也是前推后。
必要：···是前提、基础、必要的、必不可少的、不可或缺的、离不开的、必要条件。这个···就要放在箭头后面。
![[Pasted image 20240831155645.png]]

注意只有能推出来的才算，不要依赖感觉。有的A是B == 有的B是A ，注意谓语词。

![[Pasted image 20240831160504.png]]
实在题目太难就直接看选项，用答案结构来排除。 
![[Pasted image 20240831161949.png]]
要么···要么···是两种情况，或是三种。
